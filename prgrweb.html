To define a grammar and to parse a text:
First, introduce the grammar, or pick up one loading it from a grammar file, or select one from the examples. Once introduced, the grammar can also be saved in a file hitting "Save file".
Then hit "Analyze" to compile the grammar and generate the relations.
Then introduce a string to parse and hit "Parse" (a string that is "$" means no string at all, not even the empty one). If you select an example, both the grammar and the string are loaded. If the string is matched, the stack and the forest are shown.
Hitting "Parse" compiles the grammar if it has not yet been done.

To save a figure, place the pointer in it, click the right mouse button and select on the popup menu "Copy SVG" or "Copy pstricks". The figure, in the selected format is saved in the clipboard. You can then paste it in any file open in the editor of your choice.
